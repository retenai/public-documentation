    Carga de Datos

    Esta sección contiene toda la información necesaria para cargar datos desde tu sistema hacia Reten. La carga de datos es el primer paso fundamental para implementar Reten en tu negocio.

    Métodos de Conexión

    Reten ofrece dos métodos principales para cargar datos:

    Archivo

    Carga periódica de archivos CSV en bucket compartido para sincronización de datos.

    Base de Datos

    Consulta directa a base de datos expuesta por el cliente para acceso en tiempo real.

    Tipos de Datos

    Datos Maestros

    Los datos maestros representan la información fundamental de tu negocio:

    Usuarios

    Información de usuarios, contactos y direcciones.

    Transacciones

    Compras realizadas por los usuarios

    Productos

    Catálogo de productos y sus atributos

    Vendedores

    Equipo de ventas y fuerza comercial.

    Cupones

    Sistema de promociones y descuentos

    Eventos

    Acciones y actividades del sistema para análisis retrospectivo

    Configuraciones

    Las configuraciones definen cómo funciona tu negocio en Reten:

    Asignaciones

    Relaciones entre vendedores y usuarios

    Rutas

    Programación de visitas a usuarios

    Flujo de Implementación

    Para implementar la carga de datos en Reten, sigue estos pasos:

    1. Elige tu Método de Conexión

    • Archivo: Ideal para sincronizaciones periódicas y sistemas legacy
    • Base de Datos: Ideal para datos en tiempo real y sistemas modernos

    2. Prepara tus Datos Maestros

    • Define tus Productos
    • Registra tus Usuarios
    • Informa tus Vendedores en caso de tener fuerza de venta

    3. Configura tu Sistema

    • Define las Asignaciones entre vendedores y usuarios
    • Configura las Rutas de visita

    4. Implementa la Sincronización

    • Configura la frecuencia de sincronización según tu método elegido
    • Monitorea los logs y confirmaciones de carga
    • Verifica que los datos se procesen correctamente
